# Site E-commerce FashionMatch

Un site e-commerce moderne pour vendre des vêtements basé sur la personnalité de l'utilisateur.

## Fonctionnalités

- Inscription et connexion sécurisée
- Quiz de personnalité pour déterminer le style
- Recommandations de produits personnalisées
- Panier et système de commandes
- Panel administrateur pour gérer produits et commandes

## Installation

1. Cloner le repo
2. Installer les dépendances : `npm install`
3. Configurer MongoDB et mettre à jour .env.local
4. Lancer le seed : `npm run seed`
5. Démarrer le serveur : `npm run dev`

## Technologies

- Next.js
- MongoDB
- Tailwind CSS
- JWT pour l'authentification
